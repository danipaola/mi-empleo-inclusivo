# Mi Empleo Inclusivo

## Cómo funciona la actualización

La app lee las vacantes desde:

`data/vacancies.json`

Para agregar una nueva vacante y que aparezca al tocar **Actualizar**:

1. Abrí `data/vacancies.json`.
2. Copiá uno de los bloques de ejemplo.
3. Cambiá empresa, puesto, ubicación, fecha y enlaces.
4. Guardá el archivo.
5. Volvé a subirlo al mismo repositorio.
6. Abrí la app y tocá **Actualizar**.

Los estados y notas personales se guardan en el navegador y no se borran cuando se actualiza la lista.

## LinkedIn

LinkedIn no permite que una PWA común lea automáticamente publicaciones privadas o perfiles sin autorización. La app valida las publicaciones que pegues según las tres frases configuradas.
