# Mi Empleo Inclusivo — Beta completa

Esta versión incluye:

- Diseño moderno en fucsia.
- Pestañas: Vacantes, LinkedIn y Empresas.
- Tarjetas compactas.
- Logos de empresas.
- Vacantes con botones **Postularme** y **Portal oficial**.
- Checkbox **CV enviado**.
- Empresas con buscador y botón para agregar.
- LinkedIn sin foto y sin campo Puesto.
- Cada tarjeta de LinkedIn muestra:
  - Nombre
  - Última publicación
  - Botón Abrir publicación
- PWA instalable.
- Sin contadores ni cuadros blancos enormes.

## Para actualizar en GitHub Pages

1. Descomprimí este ZIP.
2. Subí todos los archivos y carpetas al repositorio.
3. Reemplazá los archivos anteriores.
4. Esperá unos minutos.
5. Cerrá y volvé a abrir la página.
6. Si seguís viendo la versión vieja, borrá los datos del sitio o desinstalá y volvé a instalar la app.
