# Mi Empleo Inclusivo

Proyecto listo para GitHub Pages.

## Cómo actualizarlo
1. Descomprimí el ZIP.
2. Subí **todos los archivos y carpetas** a tu repositorio `Empleo-inclusivo`.
3. Reemplazá los archivos anteriores.
4. Esperá unos minutos y abrí tu página.
5. En Android, cerrá y volvé a abrir la PWA. Si todavía aparece la versión anterior, borrá la app instalada y volvé a instalarla.

## Archivos editables
- `data/vacancies.json`: vacantes.
- `data/companies.json`: empresas iniciales.
- `data/linkedin_sources.json`: publicaciones iniciales.
- `data/profile_filters.json`: palabras relacionadas con tu perfil.

Las empresas y publicaciones que agregues desde la app se guardan solamente en el dispositivo mediante LocalStorage.
